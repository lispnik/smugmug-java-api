053    /**
054     * This method will upload a file to the specified album (identified by
055     * X-Smug-AlbumID) or replace an existing image (identified by X-Smug-ImageID)
056     * with the required parameters identified by the values of
057     * {@link #HTTP_HEADERS}.
058     * <p>
059     * The URL provided to methods in this class should be to the Binary-based
060     * SmugMug Upload server ({@link APIVersionConstants#BINARY_UPLOAD_SERVER_URL})
061     * or the image will not be processed.
062     * <p>
063     * This is the preferred method to upload images to SmugMug instead of the
064     * text-based methods {@link Upload} and {@link UploadFromURL}.
065     * <p>
066     * The call-structure of this class is that all the execute convenience methods
067     * will eventually delegate to {@link #execute(String, String[], byte[])} (after
068     * possibly making use of
069     * {@link #prepareUploadArgumentValues(byte[], String, Integer, Integer, String, String, String, Double, Double, Double)}
070     * to prepare arguments for them) which itself will setup the HTTP Headers with
071     * {@link #setupHTTPHeaders(PutMethod, String[])} and then eventually finish off
072     * the request by calling into
073     * {@link #executeUploadImpl(String, String[], byte[])} to perform the actual
074     * upload to SmugMug.
075     * <p>
076     * <strong>NOTE</strong>: The values for the X-Smug-ResponseType and
077     * X-Smug-Version headers will be forcibly set by all the execute methods in
078     * this class, overriding any values that were previously set.
079     * 
080     * @author Riyad Kalla
081     * @version 1.2.0
082     * @see <a href="http://smugmug.jot.com/WikiHome/API/Uploading">Binary Uploading
083     *      Doc</a>
084     */
085    public class UploadHTTPPut extends AbstractMethod {

091             * overriding any values previously set and passed in.
092             * <p>
093             * Additional convenience methods like
094             * {@link #execute(String, String, Integer, Integer, String, InputStream)}
095             * will automatically set more headers such as Content-Length and
096             * Content-MD5 after loading the image. Be sure to check the respective
097             * method's documentation to see which headers are automatically being set
098             * for you.
099             * <p>
100             * Values are: "Content-Length", "Content-MD5", "X-Smug-SessionID",
101             * "X-Smug-Version", "X-Smug-ResponseType", "X-Smug-AlbumID",
102             * "X-Smug-ImageID", "X-Smug-FileName", "X-Smug-Caption", "X-Smug-Keywords",
103             * "X-Smug-Latitude", "X-Smug-Longitude", "X-Smug-Altitude"

169            /**
170             * Used to perform a Binary-only upload using HTTP PUT.
171             * <p>
172             * Only one argument of either "X-Smug-AlbumID" (httpHeaderValues[5]) or
173             * "X-Smug-ImageID" (httpHeaderValues[6]) can be specified. If
174             * "X-Smug-AlbumID" is specified this operation will be a new-image-upload
175             * operation but if "X-Smug-ImageID" is specified the operation will be an
176             * image-replacement operation.
177             * <p>
178             * <strong>NOTE</strong>: This method forcibly sets the X-Smug-ResponseType
179             * header value to "JSON" and the X-Smug-Version header value to "1.2.0".
180             * Sub-classes are encouraged to set their own values, but this class of the
181             * API only supports these values.
182             * 
183             * @param url
184             *            The URL of the SmugMug server to communicate with. This
185             *            usually must be equal to the the SmugMug Upload URL, otherwise
186             *            the image will not be processed.
187             * @param httpHeaderValues
188             *            The HTTP header values that will be sent along with this
189             *            upload. These values represent all the upload arguments to be
190             *            associated with the uploaded image, see {@link #HTTP_HEADERS}.
191             * @param imageData
192             *            A byte array containing the actual image data that will be
193             *            uploaded.
194             * 
195             * @return the response that includes an ID for the image after it's upload
196             *         completes.
197             * 
198             * @throws IllegalArgumentException
199             *             if url is null or empty.
200             * @throws IllegalArgumentException
201             *             if both arguments "X-Smug-AlbumID" (httpHeaderValues[5]) and
202             *             "X-Smug-ImageID" (httpHeaderValues[6]) have values specified
203             *             for the headers. Only 1 value at a time can be specified.
204             * @throws NetworkException
205             *             if anything other than a HTTP status code of 200 (OK) is
206             *             returned from the server. This is not the same as a network
207             *             exception and normally represents a failure on the SmugMug
208             *             server side.
209             * @throws NetworkException
210             *             if any network-based error occurs while trying to communicate
211             *             with the SmugMug server. This can wrap many different kinds
212             *             of more specific network-based exceptions. The handler can
213             *             call <code>getCause()</code> to retrieve a more specific
214             *             reason for the exception if necessary.
215             * @throws RuntimeException
216             *             if the given fileName (httpHeaderValues[7]), represented by
217             *             the X-Smug-FileName header value, cannot be property URL
218             *             encoded before being sent to SmugMug in the upload operation.
219             * 
220             * @see #HTTP_HEADERS
221             */
222            public UploadHTTPPutResponse execute(String url, String[] httpHeaderValues,
223                            byte[] imageData) throws IllegalArgumentException,
224                            NetworkException, RuntimeException {
225                    if (!APIVersionConstants.BINARY_UPLOAD_SERVER_URL.equals(url))
226                            logger
227                                            .warn(
228                                                            "url [{}] should normally be equal to the Binary SmugMug Upload Server URL (defined by APIVersionConstants.BINARY_UPLOAD_SERVER_URL), otherwise the uploaded image won't be processed.",
229                                                            url);
230    
231                    /*
232                     * These values have to be forcibly set here and not only down in
233                     * prepareUploadArgumentValues, because this method never makes a call
234                     * to prepare the headers, and it's feasible to think that this method
235                     * could get called with invalid values set for X-Smug-Version and
236                     * X-Smug-ResponseType, so they need to be corrected right here in this
237                     * method. We also log a warning to provide feedback to the caller if
238                     * that is the case.
239                     */
240    
241                    /* This class is written to support the 1.2.0 JSON API, so set it */
242                    if (!"1.2.0".equals(httpHeaderValues[3])) {
243                            logger
244                                            .warn("X-Smug-Version, httpHeaderValues[3], was not set to a value of '1.2.0' which is required when using this method implementation. The correct value has been set, overriding the previous value.");
245                            httpHeaderValues[3] = "1.2.0";
246                    }
247    
248                    /* This can never be something different */
249                    if (!"JSON".equals(httpHeaderValues[4])) {
250                            logger
251                                            .warn("X-Smug-ResponseType, httpHeaderValues[4], was not set to a value of 'JSON' which is required when using this method implementation. The correct value has been set, overriding the previous value.");
252    
253                            httpHeaderValues[4] = "JSON";
254                    }
255    
256                    return new UploadHTTPPutResponse(executeUploadImpl(url,
257                                    httpHeaderValues, imageData));
258            }

338             * @param fileName
339             *            The name of the file whose stream will be read for data and

341             * @param inputStream
342             *            The {@link InputStream} to load and upload to SmugMug.
343             * @param caption
344             *            A caption for the image.
345             * @param keywords
346             *            The keywords assigned to the image.
347             * @param latitude
348             *            The latitude used to geocode the image.

358             *             if inputStream is <code>null</code>

365            public UploadHTTPPutResponse execute(String url, String sessionID,
366                            Integer albumID, Integer imageID, String fileName,
367                            InputStream inputStream, String caption, String keywords,
368                            Double latitude, Double longitude, Double altitude)

370                    if (inputStream == null)
371                            throw new IllegalArgumentException("inputStream cannot be null");

374                     * Load the image data here, because we need to pass it to prepare AND

376                     */
377                    byte[] imageData = APIUtils.readStream(inputStream);
378    
379                    /* Delegate to the main execute method after preparing the arguments */
380                    return execute(url, prepareUploadArgumentValues(imageData, sessionID,
381                                    albumID, imageID, fileName, caption, keywords, latitude,
382                                    longitude, altitude), imageData);
383            }

385            /**
386             * Used to prepare all the arguments and placing them in a
387             * <code>String[]</code> that can be passed immediately to
388             * {@link #execute(String, String[], byte[])} and contain all the correct
389             * values.
390             * <p>
391             * This method will take the given <code>imageData</code> and calculate
392             * the MD5 Sum for the given contents as well as set the content length
393             * based on the same array. The caller will load to load the image data
394             * ahead of time before calling this this method to prepare all the argument
395             * values.
396             * <p>
397             * <strong>NOTE</strong>: This method forcibly sets the X-Smug-ResponseType
398             * argument to "JSON" as this is the only response type this entire library
399             * supports.
400             * 
401             * @param imageData
402             *            A <code>byte[]</code> that represents the image data loaded
403             *            from a stream. This method doesn't load the bytes from the
404             *            stream directly because the caller needs access to the
405             *            <code>byte[]</code> as well, so it is the caller's job to
406             *            load it.
407             * @param sessionID
408             *            The logged in SessionID that represents the user's session.
409             * @param albumID
410             *            The ID of the album to upload the image into.
411             * @param fileName
412             *            The name of the file whose stream will be read for data and
413             *            uploaded.
414             * @param caption
415             *            A caption for the image.
416             * @param keywords
417             *            The keywords assigned to the image.
418             * @param latitude
419             *            The latitude used to geocode the image.
420             * @param longitude
421             *            The longitude used to geocode the image.
422             * @param altitude
423             *            The altitude (in meters) used to geocode the image.
424             * 
425             * @throws IllegalArgumentException
426             *             if <code>imageData</code> is <code>null</code>.
427             * @throws RuntimeException
428             *             if any of the image operation (calculateMD5Sum) fails as
429             *             defined in {@link APIUtils}. The thrown exception will wrap
430             *             the real exception that describes the failure.
431             */
432            protected String[] prepareUploadArgumentValues(byte[] imageData,
433                            String sessionID, Integer albumID, Integer imageID,
434                            String fileName, String caption, String keywords, Double latitude,
435                            Double longitude, Double altitude) throws RuntimeException {
436                    if (imageData == null)
437                            throw new IllegalArgumentException("imageData cannot be null");
438    
439                    String[] httpHeaderValues = new String[HTTP_HEADERS.length];
440                    String md5Sum = APIUtils.calculateMD5Sum(imageData);
441    
442                    /* Setup all the header values for the known values so far */
443                    httpHeaderValues[0] = Integer.toString(imageData.length);
444                    httpHeaderValues[1] = md5Sum;
445                    httpHeaderValues[2] = sessionID;
446    
447                    /*
448                     * The version (index 3) is set up in the appropriate execute(String,
449                     * String[], byte[]) method.
450                     */
451    
452                    /*
453                     * This can never be something different, so we set it here to make it
454                     * easier for sub-classes as well as setting it up in the main
455                     * execute(String, String[], byte[]) method for good measure
456                     */
457                    httpHeaderValues[4] = "JSON";
458    
459                    /*
460                     * One of these values will be null while the other will be a valid
461                     * value (indicating an upload or replace operation respectively).
462                     */
463                    httpHeaderValues[5] = APIUtils.toString(albumID);
464                    httpHeaderValues[6] = APIUtils.toString(imageID);
465    
466                    httpHeaderValues[7] = fileName;
467                    httpHeaderValues[8] = caption;
468                    httpHeaderValues[9] = keywords;
469                    httpHeaderValues[10] = APIUtils.toString(latitude);
470                    httpHeaderValues[11] = APIUtils.toString(longitude);
471                    httpHeaderValues[12] = APIUtils.toString(altitude);
472    
473                    return httpHeaderValues;
474            }

476            /**
477             * Used to take the given {@link PutMethod} and setup all the HTTP header
478             * name/value pairs for it before it is executed.
479             * <p>
480             * This is different from <code>AbstractMethod.setupPostParameters</code>
481             * because it is setting the HTTP header values and not the parameter
482             * values.
483             * <p>
484             * Any <code>null</code> values for <code>httpHeaders</code> or
485             * <code>httpHeaderValues</code> cause that pair of name/value to be
486             * ignored. For example, consider the pairs:
487             * <ul>
488             * <li>(1, bob)</li>
489             * <li>(2, mary)</li>
490             * <li>(3, <code>null</code>)</li>
491             * <li>(<code>null</code>, scott)</li>
492             * </ul>. The first 2 name/value pairs will be included in the HTTP PUT,
493             * but the 3rd and 4th will be ignored because either their name or value is
494             * <code>null</code>.
495             * <p>
496             * If more arguments exist than values, all the arguments without matching
497             * values will be ignored.
498             * 
499             * @param putMethod
500             *            The {@link PutMethod} that will be setup with all the
501             *            appropriate HTTP header name/value pairs in order to call the
502             *            SmugMug server.
503             * @param httpHeaderValues
504             *            The values to pass to the API call for each of the HTTP
505             *            headers.
506             * 
507             * @throws IllegalArgumentException
508             *             if putMethod is <code>null</code>.
509             * @throws IllegalArgumentException
510             *             if argumentValues is <code>null</code>.
511             * @throws IllegalArgumentException
512             *             if arguments.length &lt; argumentValues.length
513             */
514            protected void setupHTTPHeaders(PutMethod putMethod,
515                            String[] httpHeaderValues) throws IllegalArgumentException {
516                    if (putMethod == null)
517                            throw new IllegalArgumentException(
518                                            "putMethod cannot be null when attempting to setup the HTTP headers");
519    
520                    if (httpHeaderValues == null)
521                            throw new IllegalArgumentException("httpHeaderValues can be null");
522    
523                    if (HTTP_HEADERS.length < httpHeaderValues.length)
524                            throw new IllegalArgumentException(
525                                            "HTTP_HEADERS.length must be >= httpHeaderValues.length. If HTTP_HEADERS.length is larger than httpHeaderValues.length, each header without a value is ignored.");
526    
527                    for (int i = 0; i < HTTP_HEADERS.length; i++) {
528                            String httpHeader = HTTP_HEADERS[i];
529                            String httpHeaderValue = null;
530    
531                            if (i < httpHeaderValues.length)
532                                    httpHeaderValue = httpHeaderValues[i];
533                            else {
534                                    /*
535                                     * We had more arguments than values, so stop building the
536                                     * string
537                                     */
538                                    break;
539                            }
540    
541                            /* If we have valid name/value pair, add it to the HTTP POST method */
542                            if (!APIUtils.isEmpty(httpHeader)
543                                            && !APIUtils.isEmpty(httpHeaderValue)) {
544                                    logger.debug("\t\tAdding HTTP header name=[{}] value=[{}]",
545                                                    httpHeader, httpHeaderValue);
546    
547                                    putMethod.addRequestHeader(httpHeader, httpHeaderValue);
548                            }
549                    }
550            }

552            /**
553             * The core implementation used to perform a Binary-only upload using HTTP
554             * PUT.
555             * <p>
556             * Only one argument of either "X-Smug-AlbumID" (httpHeaderValues[5]) or
557             * "X-Smug-ImageID" (httpHeaderValues[6]) can be specified. If
558             * "X-Smug-AlbumID" is specified this operation will be a new-image-upload
559             * operation but if "X-Smug-ImageID" is specified the operation will be an
560             * image-replacement operation.
561             * <p>
562             * The header value X-Smug-ResponseType is set automatically to "JSON",
563             * overwriting any existing value set for it in the
564             * <code>httpHeaderValues</code> argument as this Java API only uses the
565             * SmugMug JSON API to communicate with SmugMug, so this value cannot be
566             * changed.
567             * 
568             * @param url
569             *            The URL of the SmugMug server to communicate with. This
570             *            usually must be equal to the the SmugMug Upload URL, otherwise
571             *            the image will not be processed.
572             * @param httpHeaderValues
573             *            The HTTP header values that will be sent along with this
574             *            upload. These values represent all the upload arguments to be
575             *            associated with the uploaded image, see {@link #HTTP_HEADERS}.
576             * @param imageData
577             *            A byte array containing the actual image data that will be
578             *            uploaded.
579             * 
580             * @return a JSON-formatted reply from the SmugMug JSON API.
581             * 
582             * @throws IllegalArgumentException
583             *             if url is null or empty.
584             * @throws IllegalArgumentException
585             *             if both arguments "X-Smug-AlbumID" (httpHeaderValues[5]) and
586             *             "X-Smug-ImageID" (httpHeaderValues[6]) have values specified
587             *             for the headers. Only 1 value at a time can be specified.
588             * @throws NetworkException
589             *             if anything other than a HTTP status code of 200 (OK) is
590             *             returned from the server. This is not the same as a network
591             *             exception and normally represents a failure on the SmugMug
592             *             server side.
593             * @throws NetworkException
594             *             if any network-based error occurs while trying to communicate
595             *             with the SmugMug server. This can wrap many different kinds
596             *             of more specific network-based exceptions. The handler can
597             *             call <code>getCause()</code> to retrieve a more specific
598             *             reason for the exception if necessary.
599             * @throws RuntimeException
600             *             if the given fileName (httpHeaderValues[7]), represented by
601             *             the X-Smug-FileName header value, cannot be property URL
602             *             encoded before being sent to SmugMug in the upload operation.
603             * 
604             * @see #HTTP_HEADERS
605             */
606            protected String executeUploadImpl(String url, String[] httpHeaderValues,
607                            byte[] imageData) throws IllegalArgumentException,
608                            NetworkException, RuntimeException {
609                    logger.debug("Executing {} Using Service URL {}", getMethodName(), url);
610    
611                    if (APIUtils.isEmpty(url))
612                            throw new IllegalArgumentException("url [" + url
613                                            + "] cannot be null or empty");
614    
615                    if (httpHeaderValues == null
616                                    || httpHeaderValues.length != HTTP_HEADERS.length)
617                            throw new IllegalArgumentException(
618                                            "httpHeaderValues cannot be null and must be the same size as HTTP_HEADERS array, where each value maps to an argument in the same position from the HTTP_HEADERS array");
619    
620                    if (httpHeaderValues[5] != null && httpHeaderValues[6] != null)
621                            throw new IllegalArgumentException(
622                                            "Only 1 value, X-Smug-AlbumID (httpHeaderValues[5] - used to specify an upload album) or X-Smug-ImageID (httpHeaderValues[6] - used to specify an image to replace), can be specified at one time, but both are currently specified.");
623    
624                    String fileName = httpHeaderValues[7];
625                    String encodedFileName = null;
626    
627                    if (APIUtils.isEmpty(fileName))
628                            throw new IllegalArgumentException(
629                                            "X-Smug-FileName (httpHeaderValues[7]) header value cannot be null or empty, it must be the file name of the image being uploaded (e.g. SmugMug.jpg)");
630    
631                    try {
632                            encodedFileName = URIUtil.encode(fileName, null);
633                    } catch (URIException e) {
634                            throw new RuntimeException(
635                                            "Unable to URL-encode the given fileName (X-Smug-FileName (httpHeaderValues[7]) header value) ["
636                                                            + fileName
637                                                            + "] in order to use it to perform the image upload operation.");
638                    }
639    
640                    String response = null;
641                    PutMethod putMethod = null;
642    
643                    try {
644                            /* SmugMug upload format is (Upload URI)/(File Name) */
645                            putMethod = new PutMethod(url + encodedFileName);
646    
647                            /* Setup the HTTP headers */
648                            putMethod.setRequestHeader("User-Agent", APIConstants.USER_AGENT);
649                            setupHTTPHeaders(putMethod, httpHeaderValues);
650    
651                            logger.trace("\tExecuting HTTP PUT...");
652    
653                            /* Set the body of the PUT to the byte data from the image */
654                            putMethod.setRequestEntity(new ByteArrayRequestEntity(imageData));
655    
656                            /* Perform the communication with the server */
657                            int code = APIConstants.HTTP_CLIENT.executeMethod(putMethod);
658    
659                            logger.debug("\tReceived HTTP status code {}", Integer
660                                            .valueOf(code));
661    
662                            /*
663                             * Make sure the server responded with a status code of 200 (OK),
664                             * otherwise we won't have a valid response from it to parse.
665                             */
666                            if (code == HttpStatus.SC_OK) {
667                                    /* Read the response from the server */
668                                    response = IOUtils.toString(
669                                                    putMethod.getResponseBodyAsStream(), putMethod
670                                                                    .getResponseCharSet());
671    
672                                    logger.debug("\tRead response, was {} bytes long", Integer
673                                                    .valueOf(response.length()));
674                            } else {
675                                    NetworkException ne = new NetworkException(
676                                                    "An HTTP status code of ["
677                                                                    + code
678                                                                    + "] was returned from the server, but a status code of 200 (OK) was expected. Something may be wrong with the SmugMug server at the moment or the network path to the SmugMug server.");
679                                    logger.error("A network error occured", ne);
680                                    throw ne;
681                            }
682                    } catch (Exception e) {
683                            RuntimeException ne = new RuntimeException(e);
684                            logger
685                                            .error(
686                                                            "An error occured while trying to execute the HTTP PUT call",
687                                                            ne);
688                            throw ne;
689                    } finally {
690                            logger.trace("\tAttempting to cleanup network resources...");
691    
692                            /*
693                             * We must release the method no matter what, otherwise the
694                             * connection is not released back to the HttpConnectionManager. Do
695                             * this as safely as possible.
696                             */
697                            if (putMethod != null) {
698                                    try {
699                                            putMethod.releaseConnection();
700                                            logger.trace("\t\tNetwork resources cleaned up!");
701                                    } catch (Exception e) {
702                                            RuntimeException re = new RuntimeException(
703                                                            "Unable to release the connection used by the HttpClient instance. This should not happen.",
704                                                            e);
705                                            logger
706                                                            .error(
707                                                                            "Unable to release the connection used by the HttpClient instance",
708                                                                            re);
709                                            throw re;
710                                    }
711                            }
712                    }
713    
714                    logger.trace("\tReturning JSON response to caller...");
715                    return response;
716            }

718            /**
719             * Class used to represent the response for a Binary Upload (HTTP PUT)
720             * method call.
721             * 
722             * @author Riyad Kalla
723             * @version 1.2.0
724             */
725            public class UploadHTTPPutResponse extends AbstractResponse {
726                    private Integer imageID;
727    
728                    private String imageKey;
729    
730                    /**
731                     * Construct a response by parsing the necessary values out of the JSON
732                     * response text.
733                     * 
734                     * @param responseText
735                     *            The JSON-formatted response text that came back from the
736                     *            SmugMug API call.
737                     * 
738                     * @throws RuntimeJSONException
739                     *             if an error occurs while parsing the JSON response text.
740                     */
741                    public UploadHTTPPutResponse(String responseText)
742                                    throws RuntimeJSONException {
743                            /* Parse base response and any error if necessary */
744                            super(responseText);
745    
746                            /* If there was an error, or the message is empty, do nothing */
747                            if (isError() || APIUtils.isEmpty(responseText))
748                                    return;
749    
750                            JSONObject responseObject = null;
751    
752                            try {
753                                    responseObject = new JSONObject(responseText);
754    
755                                    if (!responseObject.isNull("Image")) {
756                                            JSONObject imageObject = responseObject
757                                                            .getJSONObject("Image");
758    
759                                            imageID = JSONUtils.getIntegerSafely(imageObject, "id");
760                                            imageKey = JSONUtils.getStringSafely(imageObject, "Key");
761                                    }
762                            } catch (JSONException e) {
763                                    RuntimeJSONException rje = new RuntimeJSONException(e);
764                                    logger.error("An error occured parsing the JSON response", rje);
765                                    throw rje;
766                            }
767                    }
768    
769                    @Override
770                    public String toString() {
771                            return UploadHTTPPutResponse.class.getName() + "[isError="
772                                            + isError() + ", imageID=" + getImageID() + ", imageKey="
773                                            + getImageKey() + "]";
774                    }
775    
776                    /**
777                     * Used to get the ID of the image that was uploaded.
778                     * 
779                     * @return the ID of the image that was uploaded.
780                     */
781                    public Integer getImageID() {
782                            return imageID;
783                    }
784    
785                    /**
786                     * Used to get the security key of the uploaded image.
787                     * 
788                     * @return the security key of the uploaded image.
789                     */
790                    public String getImageKey() {
791                            return imageKey;
792                    }
793            }
